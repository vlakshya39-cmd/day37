# 🧭 Task Compass – Startup Edition

An interactive HTML application that helps users understand how work flows inside a startup by exploring task ownership, workflow routing, and cross-functional collaboration.

Built as part of **Day 37** of the **#60DayGrowthChallenge** using Claude AI.

---

## 🚀 Features

- 📌 Interactive drag-and-drop gameplay
- 👥 Learn task ownership across startup roles
- 🔀 Simulate real organizational workflows
- 🤝 Explore collaboration between departments
- 📊 Organizational thinking dashboard
- 📈 Performance scoring system
- 🎉 Animated completion screen
- 💻 Fully responsive modern UI
- ⚡ Pure HTML, CSS & JavaScript (No frameworks)

---

## 🎯 Learning Objectives

This project helps users understand:

- Organizational ownership
- Accountability vs responsibility
- Workflow routing
- Cross-functional collaboration
- Startup team structure
- Decision-making processes
- Delegation strategies

## 🎮 Application Stages

### Stage 1 — Who Owns This?

Users identify the primary owner responsible for solving real startup situations.

Examples include:

- Customer bug reports
- Product prioritization
- Infrastructure issues
- Revenue decline
- Customer onboarding

---

### Stage 2 — Task Routing

Users build the correct workflow by arranging departments in the proper order.

Example workflows:

- Bug reporting
- Feature launch
- Pricing updates
- Customer requests
- Infrastructure optimization

---

### Stage 3 — Collaboration Challenge

Complex business situations require multiple departments to work together.

Users decide:

- Who should be involved
- Collaboration order
- Organizational communication flow

---

## 📊 Dashboard

The application evaluates users in four categories:

- Ownership
- Delegation
- Collaboration
- Workflow Understanding

It then provides personalized organizational insights.
